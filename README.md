# Setup

1. Clone the repository

2. Create environment file

   ```bash
   cp .env.example .env
   ```

   It can remain unchanged.

3. Install dependencies

   ```bash
   pnpm install
   ```

   If you don’t have pnpm installed, you can install pnpm from https://pnpm.io/

4. Start the development server

   ```bash
   pnpm run dev
   ```
